# What is Vectorshift?

**VectorShift is a platform that allows users to build AI applications using a combination of a no-code interface and a code SDK. It offers a framework for building AI solutions such as chatbots, assistants, search engines, and automation.**

# Some of it's features: 
**No-code interface:** Allows users to create applications using drag-and-drop.

**Code SDK:** Seamless IDE integration for coding.

**Live-sync:** Allows users to set up action-based triggers.

**Automation:** Allows users to automate actions such as sending a Slack message.

# Some of it's uses:

**Chatbots:** Users can build a fully functional AI chatbot and embed it into their website.

**Search:** Users can centralize and live-sync data across their applications into one knowledge base and perform search queries over all their data.

**Automation:** Users can automate workflows end to end.


# Would you like to run this code? Then follow the steps:

**1. Clone this repo in your machine, Run the frontend code by navigating to the /frontend directory and running (1) npm i and (2) npm start.**

**2. Run backend code by navigating to the /backend directory and running uvicorn main:app --reload.**
